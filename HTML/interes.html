Variables

Las variables son contenedores para almacenar valores en un programa. En la mayoría de los lenguajes de programación, se necesita declarar una variable antes de usarla.

Funciones

Las funciones son bloques de código que realizan una tarea específica. Permiten reutilizar código y facilitan la organización y mantenimiento del programa.

Bucles

Los bucles permiten repetir una sección de código varias veces. Los bucles más comunes son el bucle for y el bucle while.

Condicionales

Los condicionales permiten ejecutar diferentes secciones de código según una condición. Los más utilizados son if, else if y else.

Arrays

Los arrays son estructuras de datos que almacenan colecciones de elementos. Pueden contener elementos del mismo tipo o de tipos diferentes.

Objetos

Los objetos son entidades que representan cosas del mundo real o conceptos abstractos. Un objeto tiene propiedades y métodos que le dan características y comportamiento.

Clases

Las clases son plantillas para crear objetos. Definen las propiedades y métodos que los objetos de esa clase tendrán.
